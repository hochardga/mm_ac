# Dead Air at Larkspur House

A staged Ashfall case package built around three clean discoveries:

1. The official electrocution story is impossible because Studio B was dark and de-powered.
2. A specific founder interview reel vanished from Mara Venn's tote.
3. Jonas Hart killed Mara to keep the board from hearing a recording that would block the sale of Larkspur House.

## Included

- `manifest.json`
- `protected.json`
- `evidence/` authored artifacts
- `evidence-generation-guide.md` with low-complexity prompts and constraints
- placeholder PNGs for the two photo artifacts

## Implementation note

Replace the placeholder PNGs with final AI-generated images using the prompts in `evidence-generation-guide.md` if you want fully polished visuals.
